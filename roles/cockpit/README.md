# Ansible Role linuxfabrik.lfops.cockpit

This role simply removes all `cockpit*` packages from the system.

Runs on

* Fedora Server 35
* RHEL 8 (and compatible)
* RHEL 9 (and compatible)
* Ubuntu 16


## Tags

| Tag       | What it does                 |
| ---       | ------------                 |
| `cockpit` | Removes all cockpit packages |
