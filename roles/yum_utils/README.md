# Ansible Role linuxfabrik.lfops.yum_utils

This role simply installs `yum-utils` package.

Runs on

* RHEL 7 (and compatible)
* RHEL 8 (and compatible)
* Fedora Server 35


## Tags

| Tag         | What it does                            |
| ---         | ------------                            |
| `yum_utils` | Installs yum-utils or dnf-utils package |
