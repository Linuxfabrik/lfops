# Ansible Role linuxfabrik.lfops.yum_utils

This role installs the `yum-utils` or `dnf-utils` package.


*Available since LFOps `2.0.0`.*


## Tags

`yum_utils`

* Installs the `yum-utils` or `dnf-utils` package.
* Triggers: none.
