# Ansible Role linuxfabrik.lfops.policycoreutils

This role installs the SELinux policy core python utilities.


*Available since LFOps `2.0.0`.*


## Tags

`policycoreutils`

* Installs the `policycoreutils-python-utils` package.
* Triggers: none.
